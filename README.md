# R-Peaks

Algorithm to detect R-Peaks in ECG signals.

The main code can be found in [the jupyter notebook](R-peaks.ipynb), there is also a report showing the steps and theory behind it in the [R-Peaks PDF](report/r-peaks.pdf) or the [main PDF](report/main.pdf). 

The MIT-BIH Arrhythmia Database was used, make sure to have it downloaded and added into a folder called databases. Later another database will be used, the ECGRDVQ and ECGDMMLD from PhysionNet.
