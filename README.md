# R-Peaks

Algorithm to detect R-Peaks in ECG signals.

## MIT-BIH Arrhythmia Database

The main code can be found in [the jupyter notebook](R-peaks.ipynb), there is also a report showing the steps and theory behind it in the [R-Peaks PDF](report/r-peaks v1.pdf). 

The MIT-BIH Arrhythmia Database was used, make sure to have it downloaded and added into a folder called databases. Later another database will be used, the ECGRDVQ and ECGDMMLD from PhysionNet.

## ECGRDVQ Dataset

For this dataset, it was first tested and developed in the [ECGRDVQ R-peaks notebook](ECGRDVQ R-peaks.ipynb), then it was extended to test in more records and compare it with Pan-Tompkins method in [ECGRDVQ results](ECGRDVQ results.ipynb). A new report was also created and can be found in the [ECGRDVQ PDF](report/ECGRDVQ R-Peaks.pdf)
